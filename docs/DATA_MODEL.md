# 元データ（Googleスプレッドシート）データモデル

対象: 柿川亭（油そば専門店）＋アブララボ（油ラボ／新店・現状データ薄い）の会計集計。
Sheet ID: `1UXr4m54rClrJknIn_o3DRupMQbYRriSXmykAkJkQx90`（owner: kakigawatei@gmail.com）
現状: 紙帳票をスキャナ読取→Excel/スプレッドシートに手入力。常に更新される。単位=円。

## タブ構成

### 設定
- 対象年(2026)/対象月(7)
- anydeli客数換算単価: 1,000円（anydeli売上÷単価=推定来店人数へ加算）
- クーポン1枚の金額: 1,000円（1枚=売上1,000円・来店1人換算）
- 標準税率10% / 軽減税率8%
- 店舗リスト: 柿川亭, アブララボ
- 仕入先リスト: めんつう/ホクショク/中越鶏卵/マルヤマ包装/竹内農園/アルベーロ/スーパー/北陸ガス/東北電力/その他
- メニュー分類: 並/大/特/焼豚並/大/特/極み/グルテンフリー/チャーシュー丼/ライス/ギフト/季節商品/その他

### 日次_柿川亭 / 日次_アブララボ（店舗別・日単位）
列: 日付, 曜日, 券売機売上, anydeli売上, クーポン枚数, クーポン売上, 通販・その他, 売上合計, 券売機杯数, anydeli換算人数, クーポン換算人数, 推定来店客数, 差額確認, 備考
- 末尾に「月次小計」行（券売機売上/anydeli/クーポン/通販/売上合計/推定来店客数）
- 例(柿川亭 7月): 売上合計 976,490円 / 推定来店客数 1,741人（7/16まで入力、7/17以降0）
- アブララボは現状すべて0（これから稼働）

### メニュー_柿川亭（券売機レシート全項目・日単位の販売数）
列: 日付 + 約40メニュー列 + 純販売数合計, 取消数, 発行数合計, 券売機売上, 備考
メニュー列例: 油そば並/大/特盛り, 味変(煮干し/塩/塩+レモン/魚介かつお), 豚骨のたれ, 焼豚油そば並/大/特, トッピング(半熟卵/ネギごま/カレースパイス/明太子マヨ/マスタード/バジル/キムチ/チーズ/にんにく辛味噌/まーらーじゃん/長岡野菜マヨ/50円), ギフトセット, 極焼豚, 背脂, グルテンフリー, チャーシュー丼, ライス, 冷やし油そば各種, ペイフォワード(100/300/1000円), クラフトコーラ, 青汁炭酸, ソフトドリンク
- 「発行－取消＝純販売数」で集計
### メニュー_アブララボ
列: 日付, 店舗, メニュー1..13, 券売機杯数（項目は券売機レシート確認後に設定・現状未設定）

### 仕入経費（伝票/レシート 1件=1行）
列: 日付, 店舗, 仕入先・費目, 税区分(税抜/税込), 税率(8%/10%), 入力額, 税抜金額, 消費税, 税込金額, 支払方法, 摘要
- 税区分: 業者伝票=税抜, スーパー等レシート=税込。税率は行ごと選択。

### 人件費（タイムカード）
列: 日付, 店舗, 氏名, 出勤, 退勤, 休憩（時間）, 勤務時間, 時給, 人件費, 備考

### 月次集計（指標 × 柿川亭/アブララボ/2店舗合計）
指標: 券売機売上, anydeli売上, クーポン売上, 通販・その他, 売上合計, 推定来店客数, 仕入・経費(税込), 人件費, 総経費, 簡易利益(売上-総経費), 売上÷推定客数(客単価)
- 共通経費は2店舗へ50%ずつ配賦する運用
- 例(7月): 柿川亭 売上976,490 / 仕入264,227 / 簡易利益712,263 / 客単価561円。アブララボ 仕入19,778 / 利益-19,778。
